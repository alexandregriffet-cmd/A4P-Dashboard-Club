# A4P Dashboard Club

Pack statique prêt à déployer sur GitHub Pages.

## Contenu
- `index.html` : dashboard club
- `equipe.html` : vue équipe
- `joueur.html` : vue joueur
- `data/joueurs.json` : base exemple
- `assets/logo-a4p.png` : logo A4P

## Déploiement
1. Créer un dépôt GitHub vide.
2. Déposer tout le contenu du pack à la racine du dépôt.
3. Activer GitHub Pages sur la branche principale.
4. Ouvrir `index.html`.

## Personnalisation
- Modifier `data/joueurs.json` pour brancher tes équipes.
- Changer les liens modules dans `index.html` si besoin.
